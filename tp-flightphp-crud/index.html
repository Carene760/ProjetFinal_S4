Bienvenue sur le système de gestion de prêts

Utilisez le menu ci-dessus pour accéder aux différentes fonctionnalités :

  • Ajouter un fond : Gérer les fonds disponibles pour l’établissement financier.
  • Créer un type de prêt : Définir les types de prêts avec leurs taux d’intérêt et durées.
  • Faire un prêt : Enregistrer un nouveau prêt pour un client.
  • Rembourser un prêt : Enregistrer un paiement de mensualité.
